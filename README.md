## 🍎 설계 및 문서

### 주요 기능

- 회원가입, 로그인
- 약속 인원들이 모일 중간 지점 검색
  - 비회원) 서비스 이용자 한명이 모든 인원들의 출발지 입력
  - 회원) 그룹의 호스트가 참가자들을 그룹에 초대. 참가자 각자가 자신의 출발지 입력
- 중간 지점의 주변 장소 추천
- 장소 공유
